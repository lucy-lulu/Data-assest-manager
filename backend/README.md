
# Spring Boot Project

This project is a Java-based Spring Boot application designed to manage canvas assets.

## Prerequisites

To run this project, you will need the following installed:

- **JDK (Java Development Kit)**: Version 21 or higher
- **Maven**: Version 3.9.8

### Check if you have these installed:

To check if Java and Maven are installed, run the following commands in your terminal:

```bash
java -version
mvn -v
```

### Biuld the project

```bash
mvn clean install
```

### Run application

```bash
mvn spring-boot:run
```

### Access the application

If you start the backend server, you can access it via http://localhost:8080.
